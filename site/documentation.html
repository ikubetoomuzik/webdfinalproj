Pi Server

The Basics

The Pi presents a different challenge as it is powered by an ARM processor. This is a more efficient if much less powerful architechture than the normal desktop x86 architechture.

Due to the large community around the Pi there are many OSs designed specifically to take advantage of the ARM architecture and run well. But being me I instead chose to find the Ubuntu Server ARM build so I could get some experience with that...

The install process was odd but relatively painless. There is a live ARM Ubuntu Server image that is updated and released semi-regularly. The install consists of just copying the iso file onto a sd card then inserting it into the Pi.
